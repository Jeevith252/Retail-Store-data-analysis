# 📂 Data Cleaning Process

## 1️⃣ Handling Missing Values

- Checked null values using:
  
```python
df.isnull().sum()
```

- Filled numerical missing values using median imputation

```python
df['Price Per Unit'].fillna(df['Price Per Unit'].median(), inplace=True)
```

- Filled categorical/boolean missing values using mode imputation

```python
df['Discount Applied'].fillna(df['Discount Applied'].mode()[0], inplace=True)
```

---

## 📊 Business Questions Solved

### 1️⃣ Total Revenue Generated

Determine the overall revenue generated by the store.

```sql
SELECT SUM(total_spent) AS total_revenue
FROM retail_sales;
```

---

### 2️⃣ Highest Revenue Generating Category

Identify which product category contributes the most revenue.

```sql
SELECT category,
       SUM(total_spent) AS revenue
FROM retail_sales
GROUP BY category
ORDER BY revenue DESC;
```

---

### 3️⃣ Top 5 Revenue-Generating Products

Find the products that generate the highest sales revenue.

```sql
SELECT item,
       SUM(total_spent) AS revenue
FROM retail_sales
GROUP BY item
ORDER BY revenue DESC
LIMIT 5;
```

---

### 4️⃣ Top 10 Customers by Spending

Identify the most valuable customers based on their spending.

```sql
SELECT customer_id,
       SUM(total_spent) AS total_spending
FROM retail_sales
GROUP BY customer_id
ORDER BY total_spending DESC
LIMIT 10;
```

---

### 5️⃣ Average Customer Spending

Calculate how much an average customer spends.

```sql
SELECT AVG(customer_total)
FROM
(
    SELECT customer_id,
           SUM(total_spent) AS customer_total
    FROM retail_sales
    GROUP BY customer_id
) t;
```
